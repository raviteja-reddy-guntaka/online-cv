### Local machine

* Get the repo into your machine 

[//]: # ()
[//]: # (```bash)

[//]: # (git clone https://github.com/sharu725/online-cv.git)

[//]: # (```)

* Install required ruby gems

```bash
bundle install
```

* Serve the site locally

```bash
bundle exec jekyll serve
```

* Navigate to `http://localhost:4000`
